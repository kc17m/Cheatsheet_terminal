    Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency. Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later.

    git init

    print working directory

    Initialize existing directory as Git repository

    git add .

    add a file for next commit

    oder auch git add [file]

    git commit -m "freetext_comment"

    commit your staged content as a new commit snapshot

    Änderungen in neues Commit verschieben

    Co-Working in Git - Link zu Beispielzenario

    git Branching

    git status

    Status in git anzeigen

    git clone /Verzeichnispfad/zum/Repository

    Repository ausschecken.

    git clone benutzername@host:/pfad/zum/repository

    wenn Remote-Repository verwendet wird

    git checkout -b branch_name

    Neuen Branch mit Namen branch_name erstellen

    git checkout main

    Zurück zum Main wechseln

    git branch -d branch_name

    Erstellten Branch branch_name wieder löschen

    git push origin branch

    Branch wird Remote-Repo hochgeladen und ist für andere verfügbar

    git pull

    Commits von Remote-Branch abrufen und mergen

    git log

    Alle Commits in History von aktuellem Branch anzeigen
